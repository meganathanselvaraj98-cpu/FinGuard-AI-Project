# Submission and Demonstration Guide

1. Extract the project ZIP.
2. Copy `.streamlit/secrets.toml.example` to `.streamlit/secrets.toml`.
3. Add the Turso URL, Read & Write token, cryptographic values, and Admin credentials.
4. Double-click `START_FINGUARD.bat`.
5. Sign in with the configured Admin account.
6. Register another normal user in a separate browser session.
7. Add profile and bank-account details for the normal user.
8. Upload `datasets/sample_transactions.csv`.
9. Demonstrate bank/statement switching, charts, budgets, predictions, health score, AI Advisor, and reports.
10. Confirm that the normal user sees only their own records.
11. Open the Admin Portal to show all-user analytics and authorized drilldown.
12. Open the Database Console to show table counts and cloud-stored records.
13. Open the Turso dashboard to confirm rows written.

Presentation description: **A secure multi-user personal-finance intelligence application with a shared cloud database, encrypted banking fields, interactive analytics, machine learning, and role-based administration.**
